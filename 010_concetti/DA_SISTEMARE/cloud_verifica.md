# PARTE A – Domande chiuse (20)

## Sezione 1 – Scelta multipla (una sola risposta corretta)

1. Il cloud computing è definito dal NIST come:
   A) Un sistema di backup remoto
   B) Un modello on-demand di accesso a risorse configurabili
   C) Un tipo di server fisico
   D) Una rete privata aziendale

2. Nel modello IaaS il cliente gestisce:
   A) Solo il codice applicativo
   B) Sistema operativo e applicazioni
   C) Solo gli utenti
   D) Solo il database

3. Nel modello PaaS il provider gestisce:
   A) Solo la rete
   B) Solo lo storage
   C) Sistema operativo e runtime
   D) Il codice dell’applicazione

4. Nel modello SaaS il cliente può:
   A) Modificare l’hypervisor
   B) Installare driver di sistema
   C) Usare un software pronto via browser
   D) Configurare la rete fisica

5. Una Availability Zone è:
   A) Un server virtuale
   B) Un gruppo di datacenter separati fisicamente
   C) Un tipo di firewall
   D) Un sistema operativo

6. L’object storage è:
   A) Un disco USB virtuale
   B) Uno storage organizzato in oggetti accessibili via API
   C) Una RAM distribuita
   D) Un database relazionale

7. Il concetto di elasticità indica:
   A) Compressione dei dati
   B) Capacità di adattare automaticamente le risorse al carico
   C) Riduzione della latenza
   D) Backup automatico

8. Nel modello di responsabilità condivisa:
   A) Il provider gestisce tutto
   B) Il cliente gestisce solo l’hardware
   C) Provider e cliente hanno responsabilità distinte
   D) Nessuno è responsabile della sicurezza

9. FaaS significa:
   A) File as a Service
   B) Function as a Service
   C) Firewall as a Service
   D) Framework as a Service

10. In IaaS, per pubblicare un sito web occorre:
    A) Solo caricare il file HTML
    B) Installare manualmente il server web
    C) Configurare il BIOS
    D) Attivare un SaaS

###### Separazione livelli: presentazione, logica, dati

###### Server: VM; Software: applicazioni isolate
###### Pubblico (provider), privato (interno), ibrido (entrambi)
###### Uso di agenti e registrazione posizione
###### Manager interroga agenti tramite SNMP
###### Riduzione hardware, flessibilità
###### Aziendale: sicura; pubblica: aperta

* rilevazione anomalie

###### Server

* DB interno

* riduzione costi

    client → app server → DB
